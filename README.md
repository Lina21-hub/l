# Literatura
Reto Literatura

Esta es una aplicación diseñada para facilitar la búsqueda, consulta y organización de libros. Permite a los usuarios buscar información sobre libros, autores, descasgas.

# Características principales
    1-Buscar libro por título
    2-Buscar libros registrados
    3-Buscar autores registrados
    4-Buscar autores vivos en un determinado año
    5-Buscar libros por idioma

# Tecnologías utilizadas

1. Java
2. SprIng Boot
3. Spring JPA
4. Base de datos: PostgreSQL5
5. API externa: https://gutendex.com/books/
